Document -> Elems

Elems   ->  Elems Elem
Elems   |   Elem

Elem    ->  Stmt
        |   TEXTO

Stmt    ->  If

If      ->  IF OPAR Cond CPAR Elems Else ENDIF
                                      | None                                             : StmtIf
                                      | ('else', elems, )                                : StmtIfElse
                                      | ('elseIf', elseIfs, elseBody : Elems | None)     : StmtIfElseIf

Else                                                ([elseifs], [else])
Else    ->  ELSE Elems                              ([], elem)
        |   ElseIf Else                             (p[1] + p[2][0], p[2][1])
        |   Empty                                   ([], [])


ElseIf  ->  ELSEIF OPAR Cond CPAR Elems             [StmtIf]
