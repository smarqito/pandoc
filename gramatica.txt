Document -> Elems

Elems   ->  Elems Elem
Elems   |   Elem

Elem    ->  Stmt
        |   Var
        |   It
        |   TEXT
        |   Nesting
        |   Newline

###############################################
#                  Stmt                       #
###############################################

Stmt    ->  If
        |  For
        |  Subtemplate

Subtemplate -> Var OPAR CPAR Pipes Newline                                                       Pipes(Subtemplate(...))

For     -> FOR OPAR Cond CPAR Newline Elems Sep ENDFOR                                           : StmtFor (Cond, Elems) 

Sep     -> SEP TEXT
        | Empty

If      ->  IF OPAR Cond CPAR Newline Elems Else ENDIF
                                      | None                                             : StmtIf
                                      | ('else', elems, )                                : StmtIfElse
                                      | ('elseIf', elseIfs, elseBody : Elems | None)     : StmtIfElseIf

Else                                                ([elseifs], [else])
Else    ->  ELSE Newline Elems                              ([], elem)
        |   ElseIf Else                             (p[1] + p[2][0], p[2][1])
        |   Empty                                   ([], [])


ElseIf  ->  ELSEIF OPAR Cond CPAR Newline Elems             [StmtIf]

Cond    ->  Var

###############################################
#                  IT                         #
###############################################

It      -> IT ItOpt Newline

ItOpt   ->  COLON Subtemplate                       Subtemplate()
        |   DOT Var                                 ItVar(p[2].getKeywords())
        |   OSQBRAC NUM COMMA NUM CSQBRAC           ItRange(p[2], p[4])
        |   Empty                                   It()

###############################################
#                  Vars                       #
###############################################

VarPipe -> Var Pipes Newline                        Pipes(p[0], p[1])   

VarAtomic ->  VarAtomic DOT ID                      p[1].add(p[2])
          |  ID                                     Var(p[1])
        
###############################################
#                  Pipes                      # 
###############################################

Pipes   -> Pipes SLASH ID                                p[0] = p[1] + [p[3]]   
        | Pipes SLASH ID NUM QUO TEXT QUO QUO TEXT QUO   p[0] = p[]                                          
        | Empty                                          p[0] = []        

###############################################
#                  Nesting                    #
###############################################

Nesting ->  Var TEXT '^' NestElems '^' Newline

NestElems -> NestElems
          |  NestElem

NestElem  -> Var
          |  TEXT

Newline   -> NL
          | Empty

#                  LIXO                       #
###############################################

CondVar ->  CondVar . CID                            p[1].add(p[2])
        |   CID                                      CondVar(p[1])
